## 🚀 部署建议

### 方案 1：使用 Cloudflare Pages（推荐）

1. **访问 Cloudflare Dashboard**
   ```
   https://dash.cloudflare.com
   ```

2. **连接 GitHub 仓库**
   - Pages → Create a project → Connect to Git
   - 选择仓库：`sxyseo/triz`

3. **配置构建设置**
   ```
   构建命令: (留空或使用 quarto render)
   构建输出目录: _site
   根目录: /
   ```

4. **环境变量**（可选）
   不需要设置

5. **部署**
   点击 "Save and Deploy"

### 方案 2：手动部署

```bash
# 构建网站
quarto render

# 部署到 Cloudflare Pages
wrangler pages deploy _site --project-name=triz-help
```

### 方案 3：GitHub Actions（已配置）

使用 `.github/workflows/deploy.yml` 进行自动部署，需要添加：
- `CLOUDFLARE_API_TOKEN`
- `CLOUDFLARE_ACCOUNT_ID`

- **解决方案**: 已注释相关代码，可后续添加 HTML5 audio 标签
